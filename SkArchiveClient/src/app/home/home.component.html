This web application is a skincare archive for (almost) all the products I have personally used.

The pages include:

  • Brands: A table of all the brands of which I've used their products
  • Products: A table of all the products I've used
  • Brands Categories: DTO counting the number of categories a brands' products fall under
  • Brands Moisutizer: DTO counting the number of moisturizers I've tried from each brand
